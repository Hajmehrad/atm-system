# 🚀 Quick Start

**Only 2 Commands:**

## 1️⃣ Install

```bash
pip install atm-system
```

## 2️⃣ Run

```bash
atm-system
```

**That's it!** Program starts 🎉

---

## 🎮 How to Use

After running, you'll see a menu:

```
------ ATM ------
1-Balance        (Check balance)
2-Deposit        (Deposit money)
3-Withdraw       (Withdraw money)
4-Change password (Change password)
5-Exit           (Exit)
```

### Examples

1. `Choice: 1` → `Password: 1234` → Shows balance
2. `Choice: 2` → `Deposit amount: 100000` → Balance becomes 1,100,000
3. `Choice: 5` → Program exits

---

## 💻 Use in Code

```python
from atm_system import atm_system

atm_system()  # Run the program
```

---

## ❓ Frequently Asked Questions

**Q: How to stop?**

A: Choose `5` (Exit) or press `Ctrl+C`

**Q: What's the default password?**

A: `1234`

**Q: What's the initial balance?**

A: 1,000,000

**Q: Can I change the password?**

A: Yes! Choose `4` from menu
