# 🚀 شروع سریع

**تنها 2 دستور:**

## 1️⃣ نصب

```bash
pip install atm-system
```

## 2️⃣ اجرا

```bash
atm-system
```

**بس!** برنامه شروع میشه 🎉

---

## 🎮 نحوه استفاده

بعد از اجرا، منو نمایش داده میشه:

```
------ ATM ------
1-Balance        (مشاهده موجودی)
2-Deposit        (واریز پول)
3-Withdraw       (برداشت پول)
4-Change password (تغییر رمز)
5-Exit           (خروج)
```

### مثال

1. `Choice: 1` → `Password: 1234` → موجودی نمایش داده میشه
2. `Choice: 2` → `Deposit amount: 100000` → موجودی 1,100,000 میشه
3. `Choice: 5` → برنامه خاتمه میشه

---

## 💻 استفاده در کد

```python
from atm_system import atm_system

atm_system()  # اجرا کن
```

---

## ❓ سوالات رایج

**S: چطور متوقف کنم؟**

A: منو انتخاب `5` (Exit) یا `Ctrl+C` بزن

**S: رمز پیش‌فرض چی‌ه؟**

A: `1234`

**S: موجودی اولیه؟**

A: 1,000,000 ریال

**S: می‌تونم رمز تغییر بدم؟**

A: بله! منو انتخاب `4`
